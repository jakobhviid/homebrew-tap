# jakobhviid/homebrew-tap

Personal [Homebrew](https://brew.sh) tap — prebuilt CLI tools for Linux and macOS.

```sh
brew tap jakobhviid/tap
brew install jakobhviid/tap/<tool>
```

Formulae here are generated and updated automatically by each tool's CI on push
to its `main`; they are not hand-maintained.

## Tools

| Formula | Platforms | What |
|---------|-----------|------|
| `pwtune` | Linux (x86_64, arm64) | Measure any speaker with any mic and build a PipeWire EQ profile. |
